Erdős–Straus conjecture

For every integer n ≥ 2, are there positive integers x, y, z with 4/n = 1/x + 1/y + 1/z? Believed true, verified past 10¹⁷, and open.

Status: open conjecture. This page reports an exact verifier, a fast solver, and the congruence structure of the easy/hard split — verified bounded computation, not a proof.

What was built

A complete computational treatment, exact integer/rational arithmetic throughout:

  • Checker — verify(n,x,y,z) tests 4·xyz = n·(yz+xz+xy). No floating point; it is the sole arbiter.
  • Solver — O(1) parametric identities for n even, n divisible by 3, and n ≡ 3 (mod 4); a bounded exact search (Simon's-favorite-factoring two-term split via divisors of q²) for the residual class.
  • Prime reduction — the key lever: the whole conjecture collapses to primes, and in fact to primes ≡ 1 (mod 4).
  • Mordell frontier — the six residue classes mod 840 where the difficulty genuinely lives, surfaced from the data.

The reduction to primes

Lemma (exact, unit-tested). If m | n and 4/m = 1/a + 1/b + 1/c, then 4/n = 1/(a·n/m) + 1/(b·n/m) + 1/(c·n/m).

So it suffices to solve 4/p for the smallest prime factor p of n, and scale. Since p ∈ {2, 3} and every p ≡ 3 (mod 4) have O(1) identities, the only inputs that ever reach the search are primes p ≡ 1 (mod 4) — and each is solved once and memoized, so an entire range costs one search per distinct hard prime.

The Mordell frontier

Mordell's identities solve 4/p for every residue mod 840 except the six squares {1, 121, 169, 289, 361, 529} = {1, 11², 13², 17², 19², 23²}. Combined with the prime reduction, the entire Erdős–Straus conjecture rests on primes p ≡ 1, 11², 13², 17², 19², 23² (mod 840). Below 10⁶ there are 2,370 such primes; all are solved by search here. Pushing the identity library to cover the non-frontier residues — shrinking the searched set toward exactly those six classes — is the next refinement.

The easy/hard congruence split

The conjecture's difficulty is concentrated by residue class. Modulo 12, only two classes have no O(1) identity and fall through to search:

Residue r (mod 12)Settled by
0, 2, 4, 6, 8, 10 (even)identity (m, m+1, m(m+1))
3, 9 (÷3)identity (m, 3m+1, 3m(3m+1))
7, 11 (≡ 3 mod 4)identity (k, nk+1, nk(nk+1))
1, 5bounded search (n ≡ 1 mod 4, coprime to 3)

This reproduces the classical reduction: 4/n is trivially solvable unless n lies in those hard residues — the ones the conjecture genuinely rests on. A worked witness: 4/5 = 1/2 + 1/4 + 1/20; 4/9973 = 1/2494 + 1/8290888 + 1/103108227451928.

Next iterations

Per the methodology: push the identity library to cover more of the hard residues mod 840, then search for a new infinite congruence family that shrinks the residual prime set — the most plausible publishable artifact, short of the (open) conjecture itself.
